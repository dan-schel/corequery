Factory reset?

If things aren't working as expected, resetting the app and starting fresh can help.

The technical details:

When you click "Factory reset", it will:

  1. Unregister all service workers for this web app
  2. Clear all local storage entries for this web app
  3. Then, navigate you back to the homepage

This will force the web app to fetch the latest copy of the code and critical data, rather than falling back to using the locally stored copies on your device.

In normal circumstances, the code should be able to update itself, but if there's a bug preventing it from doing so, this reset might be required to purge the broken code :/
